# Corporate Training & Assessment Management System

## Project Overview

The Corporate Training & Assessment Management System is a web-based application designed to manage employee onboarding, training programs, assessments, and performance reviews within an organization.

The system supports three user roles:

* Admin (HR)
* Trainer
* Employee/Intern

## Features

### Admin

* View Admin Dashboard
* Add and onboard employees
* Assign departments
* Assign trainers
* Track employee training details

### Trainer

* View Trainer Dashboard
* Manage training modules
* Create assessments and quizzes
* Submit weekly performance reviews

### Employee

* View Employee Dashboard
* Track course progress
* View score history
* View trainer feedback
* Access assigned modules

## Modules

### 1. Employee Onboarding

* Admin can add employees and assign trainers.
* Employees can view onboarding progress and update profile information.

### 2. Training Module Management

* Trainers can create training modules and assign learning materials.
* Employees can view assigned modules and track progress.

### 3. Assessment & Quiz Module

* Trainers can create quizzes and assessments.
* Employees can attempt quizzes and view results.

### 4. Weekly Performance Review

* Trainers can submit ratings, comments, and improvement areas.
* Employees can view performance feedback.
